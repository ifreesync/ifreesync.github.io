shutdown -r now 重启
shutdown -r 10 10分钟之后重启
shutdown -r 11:00 定时重启（11:00）
reboot 重启
init 6 重启
init 0 立刻关机
telinit 0 关机
poweroff 立刻关机
halt 关机
sync buff数据同步到磁盘
logout 退出登录shell
